# 店小二 Windows POS 系統使用手冊

歡迎使用 店小二 Windows POS 系統！本系統是一套完整的點餐與營運管理軟體，讓您輕鬆管理門市的每日營運。

## 📋 主要功能

### 1️⃣ 登入與啟用

- 使用帳號密碼登入系統
- 首次使用需啟用 POS 設備
- 支援多帳號切換

### 2️⃣ 點餐下單

- **快速點餐** - 直觀的點餐介面
- **商品分類** - 依分類瀏覽商品
- **商品規格** - 選擇商品大小、口味等選項
- **多種點餐模式** - 支援不同的操作習慣
- **訂單修改** - 下單前可隨時修改

### 3️⃣ 訂單管理

- **訂單查詢** - 查看今日所有訂單
- **訂單詳情** - 查看每筆訂單的完整明細
- **訂單搜尋** - 快速搜尋特定訂單
- **訂單報表** - 統計銷售數據

### 4️⃣ 付款結帳

- **信用卡付款** - 使用刷卡機進行刷卡交易
- **現金付款** - 記錄現金交易
- **多種支付方式** - 靈活支持各種付款方案

### 5️⃣ 列印與發票

- **訂單列印** - 列印廚房單與收據
- **發票列印** - 自動產生並列印發票
- **多台印表機支援** - 可連接多台印表機
- **列印設定** - 自訂列印範本與內容

### 6️⃣ 日常營運

- **桌位管理** - 管理用餐桌位狀態
- **交班記錄** - 記錄每日交班資料
- **系統日誌** - 查看系統運作紀錄
- **離線運作** - 網路斷線仍可繼續營運

## 🚀 快速開始

### 第一次使用

1. 啟動 DS2 Windows POS 應用程式
2. 輸入帳號與密碼登入
3. 系統會自動下載最新的商品與設定
4. 等待初始化完成，即可開始使用

### 日常操作流程

**典型的訂單流程：**

1. 點選「**新增訂單**」按鈕
2. 選擇商品與規格
3. 確認訂單內容
4. 選擇付款方式（信用卡 / 現金）
5. 付款完成後自動列印收據與發票

## ⚙️ 重要功能說明

### 商品管理

- 系統會自動同步最新的商品資料
- 商品分類幫助快速查找
- 可查看商品規格與庫存狀態

### 同步與更新

- 系統每天自動下載門市與商品最新設定
- 可手動執行「同步」確保資料最新
- 網路恢復後會自動同步未上傳的資料

### 設備設定

- **印表機設定** - 新增或修改連接的印表機
- **刷卡機設定** - 設定信用卡刷卡機參數
- **系統資訊** - 查看 POS 設備編號與版本
